CSS Grid Template Areas

Grid template areas provide a visual way to define your grid layout, making complex designs easier to understand and maintain.

Basic Layout

Create a common page layout with header, sidebar, content, and footer using grid-template-areas.

.container {
  display: grid;
  grid-template-areas:
    "header header header"
    "sidebar content content"
    "footer footer footer";
  grid-template-columns: 200px 1fr 1fr;
  grid-template-rows: auto 1fr auto;
}

Complex Layout

Using grid-template-areas for more complex layouts with multiple sections.

.container {
  display: grid;
  grid-template-areas:
    "nav nav nav nav"
    "sidebar-1 main main sidebar-2"
    "sidebar-1 feature-1 feature-2 sidebar-2"
    "footer footer footer footer";
  grid-template-columns: 150px 1fr 1fr 150px;
  grid-template-rows: 60px 1fr 1fr 60px;
}

Using Empty Grid Areas

You can use a period (.) to create an empty grid area that won't be filled with content.

.container {
  display: grid;
  grid-template-areas:
    "header header header"
    "sidebar content ."
    "footer footer footer";
  grid-template-columns: 200px 1fr 200px;
  grid-template-rows: auto 1fr auto;
}

Dashboard Layout

A more complex dashboard layout using grid-template-areas.

.dashboard {
  display: grid;
  grid-template-areas:
    "header header header header"
    "sidebar widget-1 widget-2 widget-3"
    "sidebar widget-4 widget-5 widget-6"
    "sidebar main main main"
    "footer footer footer footer";
  grid-template-columns: 200px repeat(3, 1fr);
  grid-template-rows: 80px repeat(2, 150px) 1fr 60px;
}

Responsive Grid Template Areas

Use media queries to redefine grid-template-areas for different screen sizes. Resize your browser to see the layout change.

/* Desktop */ .responsive-grid {
  grid-template-areas:
    "header header header"
    "nav nav nav"
    "sidebar content ads"
    "footer footer footer";
}

/* Tablet */ @media (max-width: 768px) {
  .responsive-grid {
    grid-template-areas:
      "header header"
      "nav nav"
      "content content"
      "sidebar ads"
      "footer footer";
  }
}

/* Mobile */ @media (max-width: 480px) {
  .responsive-grid {
    grid-template-areas:
      "header"
      "nav"
      "content"
      "sidebar"
      "ads"
      "footer";
  }
}
