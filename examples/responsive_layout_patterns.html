Responsive Layout Patterns

Common responsive design patterns to create layouts that work well across different screen sizes. Each pattern adapts to viewport changes in different ways.

Mobile-First Approach

Start with a mobile design and add complexity as the viewport grows. This approach focuses on delivering the core content and functionality to mobile users first.

/* Column Drop - Mobile First */ .column-drop { display: flex; flex-direction: column; } .header { /* header styles */ } .sidebar-1 { /* sidebar 1 styles */ } .content { /* content styles */ } .sidebar-2 { /* sidebar 2 styles */ } /* Tablet (768px and up) */ @media (min-width: 768px) { .column-drop { flex-flow: row wrap; } .header { width: 100%; } .sidebar-1 { flex: 1; } .content { flex: 3; } .sidebar-2 { width: 100%; /* Full width at the bottom */ } } /* Desktop (992px and up) */ @media (min-width: 992px) { .sidebar-2 { flex: 1; width: auto; /* Reset width */ } }

Tiny Tweaks Pattern

Makes small changes to layout, typography, and image size across different screen sizes. Best for single-column layouts that just need minor adjustments.

Off Canvas Pattern

This pattern places less frequently used content (like navigation) off screen on mobile devices.

Click the "Toggle Menu" button to see the off-canvas menu slide in from the left.

On larger screens, the menu is permanently visible alongside the content.

This pattern is commonly used for responsive navigation systems.

/* Off Canvas - Mobile First */ .off-canvas { position: relative; overflow: hidden; } .menu { position: absolute; left: -200px; /* Hide off screen */ width: 200px; height: 100%; transition: transform 0.3s ease; } .content { width: 100%; transition: transform 0.3s ease; } /* Active state (when menu button is clicked) */ .off-canvas.active .menu { transform: translateX(200px); /* Slide in */ } .off-canvas.active .content { transform: translateX(200px); /* Push content */ } /* Desktop (768px and up) */ @media (min-width: 768px) { .menu { left: 0; /* Always visible */ transform: none; } .content { margin-left: 200px; width: calc(100% - 200px); } .menu-toggle { display: none; /* Hide toggle button */ } .off-canvas.active .content { transform: none; /* No need to transform */ } }
