Focus States & Accessibility

This input has a label that's only visible to screen readers. 
/* Improved focus states for accessibility */
input:focus,
textarea:focus,
select:focus {
  outline: none;
  border-color: var(--primary-color);
  box-shadow: 0 0 0 3px rgba(0, 102, 204, 0.2);
}

/* Visual indication for required fields */
input:required,
textarea:required {
  border-left: 3px solid var(--primary-color);
}

/* Accessible labels that are visually hidden */
.visually-hidden {
  position: absolute;
  width: 1px;
  height: 1px;
  margin: -1px;
  padding: 0;
  overflow: hidden;
  clip: rect(0, 0, 0, 0);
  border: 0;
}

  border-color: #ff3b30;

  display: block;
}

CSS Organization & Best Practices

CSS Methodologies

BEM (Block, Element, Modifier)
.card {}                /* Block */
.card__title {}         /* Element */
.card__image {}         /* Element */
.card--featured {}      /* Modifier */
.card__button--large {} /* Element with modifier */

BEM creates a clear relationship between parent and child elements and makes the CSS more readable and maintainable.

SMACSS (Scalable and Modular Architecture for CSS)
/* Base rules */
body, a, p, h1 {}

/* Layout rules */
.l-header, .l-sidebar, .l-grid {}

/* Module rules */
.card, .btn, .form-group {}

/* State rules */
.is-active, .is-hidden, .is-collapsed {}

/* Theme rules */
.theme-dark, .theme-light {}

SMACSS categorizes CSS rules by their purpose, creating a more organized structure.

OOCSS (Object-Oriented CSS)
/* Separate structure from skin */
.btn { /* structure */ }
.btn-primary, .btn-secondary { /* skin */ }

/* Separate container from content */
.card-title { /* Styling applies regardless of location */ }

OOCSS focuses on reusability by separating structure from appearance and container from content.

Performance Optimization

Selector Efficiency
/* Avoid */
.header ul li a { color: blue; }  /* Deep nesting */
* { margin-bottom: 10px; }        /* Universal selector */

/* Better */
.nav-link { color: blue; }        /* Direct, specific */
.content > * + * { margin-top: 10px; } /* Target what you need */
Avoid Expensive Properties
/* These properties can be performance-intensive */
.expensive {
  box-shadow: 0 0 10px rgba(0,0,0,0.5);
  text-shadow: 0 1px 1px #000;
  filter: blur(5px);
  opacity: 0.8;
  background-image: radial-gradient(circle, red, blue);
}

/* For animations, prefer these properties */
.performant {
  transform: scale(1.1);  /* Uses GPU acceleration */
  opacity: 0.8;           /* Doesn't trigger layout */
}

Cross-Browser Compatibility

/* Vendor Prefixes (can be automated with tools like Autoprefixer) */
.transform-example {
  -webkit-transform: rotate(45deg); /* Safari, Chrome */
  -moz-transform: rotate(45deg);    /* Firefox */
  -ms-transform: rotate(45deg);     /* IE */
  transform: rotate(45deg);         /* Standard */
}

/* Feature Queries */
@supports (display: grid) {
  .container {
    display: grid;
  }
}

@supports not (display: grid) {
  .container {
    display: flex;
  }
}
