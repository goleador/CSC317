Flexbox Basics

Flexbox is a one-dimensional layout method designed for laying out items in rows or columns. This guide demonstrates the key properties of flexbox.

The Flex Container

To create a flex container, set display: flex on an element. All its direct children become flex items.

.container {
  display: flex;
}

Flex Item Properties

These properties apply to the flex items themselves.

The flex-grow property defines the ability for a flex item to grow if necessary.
