# Introduction to HTML

---

## What is HTML?

* **HyperText Markup Language**
* The standard language for creating web pages
* Defines structure and meaning of content
* Not a programming language → it’s **markup**

---

## Why HTML?

* It’s the foundation of every website
* Separates **structure (HTML)** from

  * **style (CSS)**
  * **behavior (JavaScript)**
* Accessible and universal → works on any device/browser

---

## Basic Structure of a Web Page

<!DOCTYPE html>  

<html>  
  <head>  
    <title>My First Page</title>  
  </head>  
  <body>  
    <h1>Hello, World!</h1>  
    <p>This is my first web page.</p>  
  </body>  
</html>  

---

## Elements

* Building blocks of HTML
* Format: <tagname>content</tagname>
* Some are **self-closing**: <img>, <br>, <hr>

---

## Attributes

* Provide extra information about elements
* Written as name="value"
* Examples:

<a href="https://example.com">Visit Example</a> <img src="photo.jpg" alt="A description of the photo">

---

# The Top 20 HTML Tags

---

## Structure

* <html> → root of the document  
* <head> → metadata (title, CSS, scripts)  
* <title> → text in the browser tab  
* <body> → visible page content  

---

## Text

* <h1> … <h6> → headings  
* <p> → paragraph  
* <br> → line break
* <hr> → divider line  

---

## Links & Media

* <a> → hyperlink
* <img> → image
* <video> → video
* <audio> → audio

---

## Lists

* <ul> → unordered list  
* <ol> → ordered list  
* <li> → list item  

---

## Tables

* <table> → table container  
* <tr> → row  
* <td> → cell (data)  
* <th> → header cell  

---

## Forms

* <form> → input container  
* <input> → text field, checkbox, radio button
* <button> → clickable button

---

## Semantic Layout

* <header> → top of page/section  
* <nav> → navigation links  
* <main> → unique page content  
* <section> → thematic grouping  
* <article> → self-contained piece of content  
* <footer> → bottom section
